Normally, after JSON Serialization, the data is stored as a text file or in a database, but the enum values are stored in numeric form, which decreases readability when opening the JSON string. Therefore, it is common to store the enum type variable's value in a string format. In such cases, you can declare an attribute on a specific member variable as shown below.


    using Newtonsoft.Json;
        using Newtonsoft.Json.Converters;
        
        public enum CompanyType {
            Apple,
            Samsung,
        }
    
        public class Company
        {
            [JsonConverter(typeof(StringEnumConverter))]
            public CompanyType CompanyType { get; set; }
        }

 

If the member variable is in the form of an array or list, you can specify the attribute as shown below.


    using Newtonsoft.Json;
        using Newtonsoft.Json.Converters;
        
        public enum CompanyType {
            Apple,
            Samsung,
        }
    
        public class Company
        {
            [JsonConverter(typeof(StringEnumConverter))]
            public CompanyType CompanyType { get; set; }
            
            [JsonProperty(ItemConverterType = typeof(StringEnumConverter))]
            public List<CompanyType> CompanyTypes { get; set; }
        }
